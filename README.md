🧠 C&A Interactive AI System

C&A System is a browser-based experimental AI simulation inspired by The Amazing Digital Circus (TADC).

It simulates two artificial intelligence entities:

🔴 Caine — chaotic, unstable entertainment AI

🔵 Abel — calm, analytical control AI

The system is built as a fake operating system interface where users interact through terminal commands, triggering dynamic AI behavior, emotional states, glitches, and simulated awareness.

This project is NOT a real AI model — it is a scripted behavioral simulation system designed to feel alive through state changes, memory storage, and interactive responses.


⚙️ Features
Dual AI system (Caine & Abel)
Persistent memory (browser localStorage)
Emotional simulation (fear, neutral, glitch states)
Dynamic UI splitting system
Terminal-based boot interface
Fake shutdown panic system
Procedural shape generation
Awareness simulation (“Caine notices you”)
Hidden interaction logic
